Welcome to CSA Buddy

This app is for volunteers at FIRST Robotics Competition (FRC) and FIRST Tech Challenge (FTC) events in the CSA (Control System Advisor) role. It's a small app that shows a quick reference page with some helpful technical tidbits.

Note: this app was developed entirely by me with no input or cooperation with FIRST. FIRST had nothing do with this app, so no endorsement or implicit approval is implied. If you want to know more information about FIRST, visit their web site at www.firstinspires.org.
